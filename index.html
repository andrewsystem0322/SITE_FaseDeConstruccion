Introduccion
Objetivo general:
Demostrar mediante el OVI el curso de Diseño de sitios WEB.

Objetivo específico 1:
Aprender sobre GIT HUB, el cual es sistema de control de versiones y será de vital importancia a la hora de diseñar un sitio web en forma colaborativa.
Objetivo específico 2:
Enseñar que es HTML5, el cual es el “esqueleto” de la página que tendremos como objetivo diseñar mediante este lenguaje.
Objetivo específico 3:
Explicar que es el CSS3, el cual es el complemente de nuestra página a Diseñar para finalizar dichos detalles de diseño y programación.
